PyElph - Installation and runnning
----------------------------------

Before you can run the application you must install:

1. Python 2.5.4 interpreter - http://www.python.org/download/releases/2.5.4/
2. The following Python packages for Python 2.5:
	- Numpy - http://sourceforge.net/projects/numpy/files/NumPy/1.6.0/
	- PIL - http://www.pythonware.com/products/pil/#pil117
	- wxPython - https://sourceforge.net/projects/wxpython/files/wxPython/2.9.1.1/
	- Matplotlib - http://sourceforge.net/projects/matplotlib/files/matplotlib/matplotlib-1.0/

or

1. Python 2.6.6 interpreter- http://www.python.org/download/releases/2.6.6/
2. The following Python packages for Python 2.6:
	- Numpy - http://sourceforge.net/projects/numpy/files/NumPy/1.6.1/
	- PIL - http://www.pythonware.com/products/pil/#pil117
	- wxPython - http://sourceforge.net/projects/wxpython/files/wxPython/2.8.12.0/
	- Matplotlib - http://sourceforge.net/projects/matplotlib/files/matplotlib/matplotlib-1.0/



After the instalation is ready, double click on the PyElph.py and the application starts.

In folder "standards" you can find the defined weight marker standards. You can define and save any weight marker standard from PyElph GUI or by editing a .marker file.
